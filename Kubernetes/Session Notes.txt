What is Kubernetes
- Kubernetes is an open source tool to manage containers. called as Orchestration tool.

Managing : Scaling the containers (Increase / decrease the containers), managing the resources (CPU, RAM), deploy the application, automate the task of managing containers

Kubernetes architecture
 - Kubernetes cluster - A cluster is a logical group of machines (physical machine/ Virtual machines)

1 one cluster
1- Master node (Same machine/ separate VM)
1 or more worker nodes (Same machine / Separate VM)

Worker Nodes will run PODS 

1 POD = 1 or more containers  (1 container run application + other will be helper container. ex: take logs)
1 container == 1application

1 POD = 1 Container = 1 Application


DockerDesktop Kubernetes (Kubeadm tool)
1 cluster = 1 Control Plane + 1 WorkerNode


What is a POD?
  - Smallest Deployable unit (First-class Citizen)
